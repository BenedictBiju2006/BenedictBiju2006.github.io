## Overview

Megasteel Corporate Website is a modern web presence built to communicate services, company profile, and contact pathways clearly across desktop and mobile devices. I focused on clean structure, responsive layouts, and consistent UI behavior to match corporate branding and improve user trust.

## Tech Stack

- React
- TypeScript
- Vercel

## Outcome

The deployed website provided a faster, more polished digital experience and improved information accessibility for prospective customers. By using a component-based approach, future content and page updates can be shipped quickly with minimal rework.
