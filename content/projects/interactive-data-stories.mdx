## Overview

Interactive Data Stories is a modular editorial platform designed to publish rich, interactive reports. The system supports narrative scrollytelling, responsive charts, and reusable storytelling blocks.

## What I Built

I built a reusable component library with performance budgets, accessibility guards, and content-driven theming. Editors can compose long-form stories without writing code while keeping visual consistency across every page.

![Interactive charts and narrative sections in the platform.](/images/project-data-story.svg)

## Impact

The platform cut production time for feature stories and improved engagement metrics across mobile and desktop readers.
