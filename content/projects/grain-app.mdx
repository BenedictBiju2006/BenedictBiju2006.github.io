## Overview

Grain is a smart grocery planning mobile application currently in development. The goal is to help users make better shopping decisions by combining personal behavior patterns, item history, and practical reminders into one streamlined experience.

## Tech Stack

- Flutter
- Firebase

## Current Progress

The app foundation includes core user flows, backend integration planning, and data structures for preference-driven recommendations. Ongoing work focuses on refining UX decisions and building reliable sync behavior for real-world grocery usage.
