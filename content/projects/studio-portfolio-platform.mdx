## Overview

Studio Portfolio Platform is a fast, content-first website that allows teams to publish case studies using only MDX files and frontmatter.

## What I Built

I designed the information architecture, built a horizontal project gallery, and implemented timeline interactions to connect experiences with project outcomes.

![Minimal project card system and timeline components.](/images/project-studio.svg)

## Impact

New project pages can now be published by adding a single MDX file, making the platform extendable and low-maintenance.
