## Overview

Chess.com Discord Bot helps club members track players, monitor activity, and organize competition updates directly inside Discord. I implemented command flows for lookups, profile summaries, and reporting that combine API data with persisted SQL records.

## Tech Stack

- C#
- SQL
- Discord API

## Outcome

The bot centralized club operations into one channel-friendly interface and removed the need for manual spreadsheet tracking. It made player information easier to access and gave organizers a more reliable way to manage community activities.
